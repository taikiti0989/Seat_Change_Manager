# 席替えソフト

席替えをサポートするGUIアプリです。Python標準ライブラリのみで動作します。

## 起動方法

```bash
python3 main.py
```

## 使い方

### 1. 設定画面
- **列数（横）**: 横に何席あるか
- **行数（縦）**: 縦に何席あるか
- **生徒数**: クラスの生徒数（席数以下にしてください）

### 2. レイアウト編集画面
- 席をクリックすると「無効席（使わない席）」に切り替わります
- 再度クリックすると有効に戻ります
- L字型や凸型など、長方形以外の席配置もこれで対応できます

### 3. 制約ファイルの読み込み（任意）
- 「制約ファイル読込」ボタンからテキストファイルを選択します
- 仲の悪い生徒ペアを事前に指定しておくと、その生徒が隣り合わないよう最適化されます
- 完全に分離できない場合は、最も違反が少ない配置を採用します

### 4. 席替え実行
- 「席替え実行」ボタンで結果が表示されます
- 生徒は **①②③...** の丸数字で表示されます（個人情報保護）
- 「もう一度」ボタンで再抽選できます

## 制約ファイルの書き方（constraints.txt）

```
# コメント行（# で始まると無視される）
# 仲が悪い生徒を「A,B」の形式で記述
1,2
3,5
```

`constraints.txt` のサンプルが同梱されています。参考にしてください。

## 動作環境

- Python 3.8 以上
- Tkinter（Pythonに標準搭載）
